# SecureVote Prototype V1

This repository contains the minimal V1 prototype of the SecureVote system.

The purpose of this prototype is to demonstrate core voting functionality for the interim report.

Only essential features are included at this stage.

## Technologies

- **Frontend:** React + Bootstrap
- **Backend:** Flask (Python)
- **Database:** MongoDB
- **Security (Prototype):** Basic SHA-256 hashing
- **No CI/CD or Cloud Deployment yet**

## Prototype Capabilities

- User login
- Cast a vote (one vote per student)
- View vote results
- Basic navigation between screens
